### 🚀 Highlighted Projects

#### 1. [DNK. | E-Commerce Price Intelligence Engine](https://github.com/SouravEmpire07/DNK-A-Full-Stack-Real-Time-E-Commerce-Price-Intelligence-Engine)
*A premium price comparison application built to scrape and track e-commerce prices in real time.*
* **Tech Stack:** Python (Flask), SQLite, SQLAlchemy, BeautifulSoup4, Requests, JavaScript, CSS (Glassmorphism design).
* **Key Features:** Real-time web scraping fallback orchestration (DuckDuckGo Search + Bing Shopping APIs), secure user sessions (Flask-Login), featured deals catalog, and database-persisted newsletter subscriptions.

#### 2. [QuickTask Web](https://github.com/SouravEmpire07/QuickTask-Web)
*A high-performance, sleek task manager designed with a minimalist dark theme.*
* **Tech Stack:** Flask, SQLite, Jinja2 template inheritance, custom CSS, Bootstrap 5.
* **Key Features:** Fully-implemented CRUD lifecycle, real-time tags, and categorizations (General, Music, Work, Study, etc.) with dropdown filter options.

#### 3. [REST API Data Extractor & CSV Parser](https://github.com/SouravEmpire07/mastering-fullstack-and-backend)
*Automated script pipelines that fetch, structure, and export web-based API data.*
* **Tech Stack:** Python, `requests` library, REST API integrations, JSON Parsing.
* **Key Features:** Fault-tolerant connection handling, auto-creating directory hierarchies, and generating clean, formatted CSV datasets.
